# Intern Analytics Platform

## Folder Structure

```
intern_analytics/
├── app.py                        ← Entry point. Run this.
├── config.py                     ← DB config, model paths, constants
├── auth.py                       ← Login logic
├── database.py                   ← All SQL queries
├── components.py                 ← Shared UI helpers (CSS, cards, sidebar)
│
├── pages/
│   ├── manager.py                ← Manager Dashboard
│   ├── employee.py               ← Mentor Dashboard
│   ├── intern.py                 ← Intern Dashboard
│   └── ml_insights.py            ← ML Insights (Manager + Mentor)
│
├── ml/
│   ├── predictor.py              ← Loads models, runs inference
│   ├── train_classification.py   ← Train at-risk classifier (run once)
│   └── __init__.py
│
└── models/
    ├── productivity_model.pkl     ← Copy from teammate's ml/ folder
    ├── clustering_model.pkl       ← Copy from teammate's ml/ folder
    └── classification_model.pkl   ← Generated by train_classification.py
```

---

## Setup Steps

### 1. Install dependencies
```bash
pip install streamlit plotly bcrypt psycopg2-binary scikit-learn pandas numpy
```

### 2. Copy model files
Copy these from your teammate's `ml/` folder into `models/`:
- `productivity_model.pkl`
- `clustering_model.pkl`

### 3. Train the classification model
Run once to generate `classification_model.pkl`:
```bash
cd ml
python train_classification.py
```
> Update `base_dir` and `ml_dir` paths in `train_classification.py` to match where your teammate's CSV outputs are.

### 4. Configure DB (if needed)
Edit `config.py` or set environment variables:
```bash
set DB_HOST=localhost
set DB_NAME=intern_analytics
set DB_USER=postgres
set DB_PASSWORD=1234
```

### 5. Run the app
```bash
streamlit run app.py
```
Opens at http://localhost:8501

## ML Models Summary

| Model | Type | Purpose |
|---|---|---|
| `productivity_model.pkl` | Linear Regression | Predict productivity score from activity features |
| `clustering_model.pkl` | KMeans (3 clusters) | Group interns: High Performer / Consistent / Learning Phase |
| `classification_model.pkl` | Random Forest | Predict if intern is at-risk (yes/no) |
